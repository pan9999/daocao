具体具体操作过程：

1.消费者进去internet，查看企业和商家的网页；

2.消费者可以通过注册或者登录等，信息来挑选商品；

3.可以选择支付方式

4.商家或者企业的客户服务器检查支付方服务器，安全 情况

2.网上订购

​	客户在对商品了解后，可以再网页中查询价格，确定数量来丁欧商品。
